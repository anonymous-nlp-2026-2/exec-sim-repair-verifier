\section{Conclusion}
\label{sec:conclusion}

We presented the first systematic comparison of three execution-free verification formulations (\cls{}, \cwm{}, \traj{}). At function level, \cls{} and \cwm{} are statistically equivalent (TOST $p{=}0.007$, 5 seeds); \traj{} parity is model-dependent. Per-test classification achieves 217$\times$ higher throughput while providing diagnostic signal. Complementary errors (oracle: 90.63\%/95.83\%; single-seed) motivate complexity-aware routing. At repository level under ${\leq}$8B LoRA, divergence reaches 18--26\pp{} ($z{=}-23.7$, 3 seeds per model), with \cwm{} falling below majority baseline---consistent across families but dependent on multiple confounds (\S\ref{sec:threats}). Three failure modes (format collapse, decision collapse, aggregation penalty) are consistent across families. \cls{} is recommended primarily for its throughput advantage and diagnostic capability; the accuracy advantage over \traj{} is directional but underpowered ($n{=}48$, MDE${\approx}$25\pp{}; Appendix~\ref{app:power}).

Three directions follow. First, oracle gains motivate complexity-aware routing; the failure mode taxonomy provides candidate routing features. Second, the generation bottleneck is semantic---four interventions fail (Appendix~\ref{app:extended_analysis}(g))---suggesting that closing the gap requires capacity scaling or execution trace pretraining~\cite{meta2025cwm}; testing at 32B+ scale would resolve the primary open question. Third, larger-scale evaluation of per-test diagnostics for downstream repair ($n{=}44$ in our pilot) would test whether \cls{}'s throughput advantage extends to repair pipelines.

\section*{Limitations}

Repo-level \cwm{} uses 3 seeds per model (Qwen3-8B, DeepSeek-6.7B); the gap (18--26\pp{}) exceeds any single assessed confound (\S\ref{sec:threats}), but correlated confounds mean definitive causal attribution requires further replication. Our \cwm{} baseline uses ${\leq}$8B LoRA fine-tuning, not the original 32B CWM with execution trace mid-training~\cite{meta2025cwm}; at larger scale, the generation bottleneck may diminish. Training data comprises synthetic mutations~\cite{andrews2005mutation,just2014mutants}; the synthetic-to-real transfer gap for absolute accuracy remains untested, though the coupling effect supports relative ranking validity (Appendix~\ref{app:limitations}). The downsample ablation for \traj{}'s data volume advantage uses a single model, seed, and complexity level; extending to additional configurations would strengthen this control. Model coverage is limited to 6--8B code-specialized LLMs; generalization to larger or general-purpose models is untested. All code, data, and model checkpoints will be released upon acceptance.
